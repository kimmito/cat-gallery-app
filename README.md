# Тестовое задание для Учи.ру

Репозиторий содержит выполненное тестовое задание на позицию Front-end разработчика: проект "Кошачий пинтерест".

## О проекте

Приложение позволяет:
- просматривать список котиков из TheCatAPI,
- добавлять/удалять котиков в избранное,
- просматривать отдельную страницу с избранными котиками.

## Стек

- React 19
- TypeScript
- Vite
- React Router
- TanStack Query
- Tailwind CSS
- Sonner (toast)

## Что реализовано по ТЗ

- Открытие вкладки "Все котики" по умолчанию.
- Добавление и удаление котика в избранное.
- Хранение избранного на клиенте через localStorage.
- Отдельная вкладка "Любимые котики".

## Что сделано сверх ТЗ (сильные стороны)

- Адаптивная сетка карточек (1/2/3/4/5 колонок в зависимости от ширины экрана).
- Бесконечная прокрутка с автоподгрузкой через IntersectionObserver.
- Динамический размер первой загрузки: стартовая порция котиков подбирается под размер экрана, чтобы лучше заполнять видимую область.
- Корректная пагинация без API-ключа при лимите TheCatAPI (батчи без пересечений страниц).
- Дедупликация котиков по `id` при объединении страниц.
- Разделение состояний загрузки:
	- на первую загрузку показывается Loader-компонент,
	- на последующие подгрузки показывается текст "... загружаем ещё котиков ...".
- Явный Error state с кнопкой "Повторить" при неуспешной первичной загрузке.

## Запуск проекта

```bash
yarn install
yarn dev
```

Прод-сборка:

```bash
yarn build
yarn preview
```

## Скрипты

- `yarn dev` - запуск dev-сервера
- `yarn build` - typecheck + production build
- `yarn preview` - локальный просмотр production сборки
- `yarn lint` - проверка ESLint

## Переменные окружения

API-ключ для TheCatAPI опциональный.

Создай `.env` в корне проекта:

```env
VITE_CAT_API_KEY=your_api_key
```

Если ключ не указан, приложение продолжает работать в режиме без ключа (с учетом ограничений API).
